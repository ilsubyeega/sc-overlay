Customise cheatsheet: Gist
Using custom port
StreamCompanion uses 20727 ports by default. If you are using a different port, you can write the #(port) on the back of url.
Example: https://sc-overlays.ilsubyeega.com/background#1972 (1972 port)
Full Background /background
Use the background to fill the screen.
PP Odometer /ppodometer
Obtain the current pp using odometer.js** animation (Only visible at playing/result currently)
